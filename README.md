# Project List Fetcher (Go)

## Описание

Данное приложение разработано на языке Go и предназначено для получения списка проектов из внешнего API, обработки полученных данных в формате JSON и формирования итогового вывода на основе шаблона.

Доступ к API осуществляется с использованием ключа, который хранится локально в файле `secret.txt` и передаётся в заголовке HTTP-запроса.

Программа демонстрирует работу с сетевыми запросами, обработкой JSON-данных, файловой системой и шаблонизацией в Go.

---

## Цель проекта

Разработать программный модуль, обеспечивающий:

- получение данных о проектах из внешнего сервиса по HTTP
- безопасное использование API-ключа
- десериализацию JSON в структуры Go
- обработку полученных данных
- формирование итогового результата на основе шаблона
- вывод или сохранение результата

---

## Функциональные возможности

- Чтение API-ключа из локального файла
- Выполнение HTTP GET-запроса к внешнему API
- Передача ключа в заголовке запроса
- Обработка ответа сервера
- Парсинг JSON-данных
- Подготовка данных для отображения
- Генерация итогового вывода

---

## Требования

- Go версии 1.20 или выше
- Доступ к сети Интернет
- Действительный API-ключ

---

## Структура входных данных

API возвращает список проектов в формате JSON:
