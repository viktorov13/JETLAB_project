Целью проекта является разработка программного модуля на языке Go, обеспечивающего автоматическое получение данных о проектах из внешнего API, их обработку и формирование выходного шаблона на основе полученной информации.

Программа должна:
выполнять HTTP-запрос к удалённому сервису с использованием API-ключа
безопасно считывать ключ доступа из локального файла secret.txt
получать JSON-ответ со списком проектов
десериализовать данные в структуры Go
передавать данные в обработчик бизнес-логики
формировать итоговый шаблон (текстовый или HTML)
выводить или сохранять результат
Проект направлен на демонстрацию навыков работы с:
сетевыми запросами (HTTP-клиент)
обработкой JSON
управлением конфигурацией и секретами
шаблонизацией в Go
структурированием кода
